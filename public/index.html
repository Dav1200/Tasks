Task Management

How to Use the Task Management System

  • Create Task: Click the "Create Task" button to open the form. Fill in the details and submit to add a new task.
  • Search by ID: Enter a task ID in the search box and click "Go" to find a specific task.
  • Update Status: Use the "Update"button in the task table to modify the status of a task.
  • Delete Task: Click the"Delete" button in the task table to remove a task.
